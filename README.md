# ranek

## Project setup
```
yarn install
```

### Install json-server
```
npm install -g json-server
```

### Start API
```
cd ranek-api
```
```
json-server --watch ranek.json
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
